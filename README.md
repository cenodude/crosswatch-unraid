# CrossWatch for Unraid

This repository contains the official Unraid Community Applications template for CrossWatch.

CrossWatch synchronizes watchlists, ratings, history and playback progress between supported media servers and tracking services.

## Links

Application repository, https://github.com/cenodude/CrossWatch

Website, https://www.crosswatch.app/

Documentation, https://wiki.crosswatch.app/

Support, https://github.com/cenodude/CrossWatch/issues

Container image, ghcr.io/cenodude/crosswatch:latest
